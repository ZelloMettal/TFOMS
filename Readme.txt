По условию задания прямая связь указанна только между таблицами PEOPLE и HISTLPU,
поэтому связи между таблицей PEOPLE и правочниками(T001, T007) не создавались.

Для выполнения поиска было создано две хранимые процедуры. 
Для поиска по ФИО и дате рождения FindByNameAndBirthDay
Для поиска по номеру полиса процедура FindByENP
Так как обе процедуры производят поиск в рамках одной базы данных, то с целью сокращения кода 
внутри процедуры FindByENP вызывается процедура FindByNameAndBirthDay. Таким образом, 
если номер полиса был найден в базе, то по нему выводится соответствующая информации о человеке.
